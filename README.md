\# Instagram Clone

## Database Setup

1. Import `instagram_clone.sql` into MySQL
2. Database name: instagram_clone
3. Update DB credentials in `DBConnector.java`
4. Run project on Apache Tomcat (NetBeans 8.2)


Mini Instagram-style web application built using:

\- Java (JSP \& Servlets)

\- MySQL

\- HTML, CSS

\- NetBeans 8.2



Features:

* User login using hashing(SHA -256)
* proper structure of MVC
* User Authentication
